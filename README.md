## Getting Started

Edit `.env`

```
NEXT_PUBLIC_REST_API_ENDPOINT=http://192.168.1.50:4001
NEXT_PUBLIC_REST_API_ENDPOINT_DATA=http://localhost:3001
```

Run the development server:

```bash
npm run dev
npm run api
```
